# 두 수의 합 (Two Sum)

## 문제 설명
정수 배열 `nums`와 정수 `target`이 주어집니다. `answer(nums, target)` 함수를 구현하여,
`nums[i] + nums[j] == target`를 만족하는 서로 다른 두 인덱스 `[i, j]`를 반환하세요.
해당 쌍이 존재한다고 가정합니다.

## 입력 형식
- `nums`: 정수로 이루어진 리스트 (길이 2 이상)
- `target`: 정수

## 출력 형식
- 길이 2인 리스트 `[i, j]`를 반환합니다.
- `i < j`를 유지하세요.

## 예시
- 입력: `nums = [2,7,11,15]`, `target = 9`
  - 출력: `[0, 1]`
- 입력: `nums = [3,3,4]`, `target = 6`
  - 출력: `[0, 1]`

## 제약
- `2 <= len(nums) <= 10^4`
- `-10^9 <= nums[i], target <= 10^9`
- 항상 해가 하나 존재합니다.

## 풀이 아이디어 힌트
- 해시맵(딕셔너리)으로 `target - nums[i]` 값을 찾으면 O(n)에 해결할 수 있습니다.
